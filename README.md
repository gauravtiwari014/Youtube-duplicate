# Youtube-duplicate
A fully functional YouTube Clone built using modern web technologies that allows users to browse, search, and watch videos just like the original YouTube platform.
# Features

🔍 Search videos in real-time

🎥 Watch videos with a responsive player

📂 Browse categories (Trending, Music, Gaming, etc.)

👍 Like, share, and interact with videos (UI-based or functional)

📱 Fully responsive design (Mobile + Desktop)
# Tech Stack

Frontend: HTML, CSS, JavaScript / React.js

Backend: Node.js, Express.js (if used)

API: YouTube Data API v3

Database: MongoDB / Firebase (if used)

Styling: Tailwind CSS / Bootstrap
